# My Dotfiles for i3wm

## Information

OS: [Arch](https://archlinux.org/)

Terminal: [Kitty](https://sw.kovidgoyal.net/kitty/)

Font: [JetBrains Mono Nerd](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/JetBrainsMono.zip)

Color Scheme: [Kanagawa](https://github.com/rebelot/kanagawa.nvim)

Icon: [Papirus](https://github.com/PapirusDevelopmentTeam/papirus-icon-theme)

Cursor: [Bibata Modern Ice](https://www.gnome-look.org/p/1197198/)

Status Bar: [i3bar](https://i3wm.org/i3bar/) with [i3status](https://i3wm.org/i3status/)

Editor: [NeoVim](https://github.com/schmeekygeek/dotfiles/tree/main/.config/nvim)
